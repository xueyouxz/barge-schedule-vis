### **📊 详细讲解**

#### **1. 为什么选择这个航线组合？**

**船0选择 `["LTA", "TC2", "BLCT3"]`（中转）而非直航：**
- ✅ **优势**：
  - 可以装载更多货物（龙潭+太仓货）
  - 提高积载率（从50%→85%）
  - 太仓有大量去BLCT3的货
- ❌ **代价**：
  - 中转费用：约6000元（10箱×600元）
  - 额外时间：约12小时
  - 太仓码头占用

**综合评估**：中转方案得分245.7 > 直航方案得分198.3

#### **2. 为什么某些箱没有被分配？**

```python
unassigned_containers = [箱15, 箱16, 箱23]

# 未分配原因：
箱15: "ETD超期"          # 预计到达时间晚于ETD
箱16: "容量不足"          # 所有船都已满载
箱23: "危险品+无直航"     # 危险品不能中转，但无直航到其目的地
```

每个未分配的箱施加惩罚分：`-10分/箱`

#### **3. 中转的时间窗口约束**

```python
# 箱0的中转过程
船0在LTA装箱0 → 航行12.5h到TC2 → 卸货0.5h（完成时间：13h）
                                    ↓
                        太仓码头暂存箱0
                                    ↓
船2在TC2装箱0 → 等待靠泊2h → 开工准备1h（开始装货：3h）

# 时间窗口判断
船0完成卸货时间(13h) < 船2开始装货时间(3h + 46h离港时差) ✓ 可行
```

#### **4. ETD约束检查**

```python
# 箱4（ETD: 2025-08-25 1:42）
当前时间: 2025-08-16 0:00
预计到达BLCT3时间: 2025-08-16 0:00 + 48.7h = 2025-08-18 0:42
ETD剩余时间: 201.2h

判断: 48.7h < 201.2h ✓ 满足ETD约束
```

#### **5. 小箱配对逻辑**

```python
# 中转小箱必须配对（避免拆箱操作复杂性）
```
