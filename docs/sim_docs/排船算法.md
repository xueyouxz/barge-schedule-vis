## **关键数据结构说明**

1. **航线表示**：`[起始港, 港口1, 港口2, ...]`
2. **集装箱路径**：`[(起点, 终点, 船ID), ...]`，可能有多段（中转）
3. **装船计划**：`barge_plan_to_load_containers[船ID][码头] = [箱列表]`
4. **状态跟踪**：
   - `barge_status`: 每段航程的载货状态
   - `barge_timeline`: 南京船在太仓的时间线（中转时间窗口）
   - `barge_nb_unload_times`: 在宁波码头的卸货时间（ETD约束）

---

## **算法特点总结**

1. **多目标优化**：综合考虑转码头率、费用、时间、船舶成本等多个指标
2. **复杂约束**：容量约束、时间窗口约束、中转可行性约束、ETD约束
3. **组合优化**：通过航线组合+贪心分配求解NP难问题
4. **启发式加速**：基于货量分布的航线概率采样，减少搜索空间

这个算法是整个仿真系统的核心智能决策模块，决定了所有船舶的航线和集装箱的分配方案。
