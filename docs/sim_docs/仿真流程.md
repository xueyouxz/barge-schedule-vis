def handle_event():
    global current_event, event_list
    if event_list:
        current_event = heapq.heappop(event_list)
    if current_event[1] == "plan_loading_daily":
        plan_loading_daily_with_strategy(current_event)
    elif current_event[1] == "plan_loading":
        plan_loading(current_event)
    elif current_event[1] == "load":
        load(current_event)
        # 装船后从码头移除对应集装箱
        for port in ports:
            port.remove_containers(current_event, tmp_remove_containers)
    elif current_event[1] == "depart":
        depart(current_event)
    elif current_event[1] == "arrive":
        arrive(current_event)
    elif current_event[1] == "start_unload":
        start_unload(current_event)
    elif current_event[1] == "end_unload":
        end_unload(current_event)
        # 卸船后添加对应集装箱（中转）
        for port in ports:
            port.add_containers(tmp_add_containers)
    elif current_event[1] == "import_plan_loading":
        return # TODO：目前不考虑进口货排船
        import_plan_loading(current_event)
```

**详细说明：**
- 从事件堆中弹出时间最早的事件
- 根据事件类型分发到不同的处理函数
- 支持的事件类型：
  - `plan_loading_daily`: 每日排船计划
  - `plan_loading`: 单船装船计划
  - `load`: 装船
  - `depart`: 离港
  - `arrive`: 到港
  - `start_unload`: 开始卸船
  - `end_unload`: 完成卸船
  - `import_plan_loading`: 进口箱排船（当前未实现）

def end_unload(event):
    global tmp_add_containers, loading_unloading_records, barge_records, container_records
    current_time = event[0]
    barge_id, port_name = event[2], event[3]
    barge = all_barges[barge_id]
    unloaded_containers = barge["containers"].get(port_name, [])
    
    if unloaded_containers:
        # 更新集装箱状态
        for c in unloaded_containers:
            set_container_attr(c, "location", port_name)
            new_leg = get_container_attr(c, "current_leg")+1
            set_container_attr(c, "current_leg", new_leg)
            # 到达终点
            if new_leg == len(get_container_attr(c, "route")):
                set_container_attr(c, "status", "finish")
                container_records.append(c)
            # 中转箱
            else:
                set_container_attr(c, "status", "waiting")
                tmp_add_containers[port_name].append(c)
```

**详细说明：**
- 更新集装箱位置和运输段信息
- 判断是否到达终点，是则标记为 "finish"，否则加入中转队列
- 更新船舶载货信息（减去卸下的集装箱）
- 记录卸船事件
- 计算宁波港码头的转码头费用和中转费用

## **总结：完整执行流程**

1. **初始化** → 解析参数、配置场景
2. **数据加载** → 码头、船舶、集装箱数据
3. **事件初始化** → 创建每日排船计划事件
4. **主循环** → 处理事件队列中的所有事件
5. **事件处理链** → 
   - plan_loading_daily（日计划）
   - plan_loading（单船计划）
   - load（装船）
   - depart（离港）
   - arrive（到港）
   - start_unload（开始卸船）
   - end_unload（完成卸船，可能循环回 plan_loading）
6. **结果输出** → 统计分析、生成报表

整个系统是一个**离散事件驱动**的仿真框架，通过事件队列按时间顺序模拟驳船运输的完整生命周期。
